# PWA Icons Placeholder

Place the following icon files in this directory:

1. **icon-192.png** - 192x192px PNG file
   - Thinkior brand logo
   - Purple background (#534AB7)
   - White or light-colored logo text/icon

2. **icon-512.png** - 512x512px PNG file
   - Same design as icon-192.png
   - Used for splash screen and larger displays

## Recommended Design:
- Background: #534AB7 (purple)
- Text: "L" or Thinkior logo in white
- Style: Clean, modern, recognizable at small sizes

## Tools to Create Icons:
- Figma
- Canva
- Adobe Express
- Online PWA icon generators

Once created, ensure both files are PNG format and placed in this directory.
